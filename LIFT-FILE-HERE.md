# Place your dictionary.lift file here

This is where your source .lift dictionary file should be placed.

The file should be named `dictionary.lift` and placed in the root of this project.

## What is a .lift file?

.lift (Lexicon Interchange FormaT) is an XML-based format used by linguistic software like SIL FieldWorks for storing dictionary data. It contains:
- Lexical entries (words)
- Definitions and glosses
- Example sentences
- Grammatical information
- Multiple writing systems
- And more!

## Getting started

1. Place your `dictionary.lift` file in this directory
2. Run `npm install` to install dependencies
3. Run `npm run build:data` to process the .lift file
4. Run `npm run dev` to start the development server
